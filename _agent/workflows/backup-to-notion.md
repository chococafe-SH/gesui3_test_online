# /backup-to-notion ワークフロー

このワークフローは、現在のセッションで作成された全ての成果物（アーティファクト）をNotionの「成果物バックアップ」データベースへ完全に同期します。

## 実行手順

1. **成果物の確認**:
    - `brain/<conversation-id>/` 内にある全ての `.md` ファイルを確認します。
2. **内容の読込**:
    - 全てのファイルについて、`view_file` を使用して内容を全文取得します。
3. **Notionへの書き出し**:
    - `notion_full_backup` Skillの指示に従い、各ファイルをNotion上の個別のブロック、または1つのページ内のセクションとして詳細に書き出します。
    - **絶対に要約せず、全てのテキストを転記してください。**
4. **完了報告**:
    - 書き出しが完了したNotionページのURLをユーザーに共有します。
